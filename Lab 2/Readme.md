# Cấu trúc lệnh cơ bản - Câu lệnh if... elif...else
- Câu lệnh if... elif...else là các câu lệnh nối đuôi
- Cú pháp: 
- if điều kiện 1:
    khối lệnh bên trong
- elif điều kiện 2:
    khối lệnh bên trong
- ...
- else:
    khối lệnh bên trong
- Các điều kiện sẽ được kiểm tra theo thứ tự từ đầu đến cuối. Nếu có 1 điều kiện thỏa mãn thì nó sẽ chạy xong điều kiện đó và sẽ ngừng
- Các câu lệnh này có thể lồng nhau
